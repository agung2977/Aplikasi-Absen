# Aplikasi-Absen

Program di atas merupakan aplikasi absensi kelas berbasis GUI (Graphical User Interface) yang dibuat menggunakan library tkinter pada Python. Aplikasi ini memungkinkan pengguna untuk memasukkan nama siswa dan memilih keterangan kehadiran seperti Hadir, Izin, Sakit, atau Alfa melalui tampilan yang sederhana dan mudah digunakan. Setiap data yang dimasukkan akan divalidasi terlebih dahulu untuk memastikan nama dan status tidak kosong. Setelah itu, sistem secara otomatis mencatat waktu absensi berdasarkan jam saat input dilakukan menggunakan modul datetime. Data absensi yang telah diinput akan ditampilkan pada daftar (Listbox) di dalam aplikasi dan disimpan sementara dalam sebuah list Python.

Selain menampilkan data di layar, aplikasi ini juga memiliki fitur penyimpanan ke dalam file berformat CSV menggunakan modul csv. File akan diberi nama otomatis sesuai tanggal saat penyimpanan dilakukan, sehingga setiap hari memiliki file absensi yang berbeda. Format penyimpanan menggunakan tanda titik koma (;) sebagai pemisah kolom agar dapat langsung terbaca rapi di Microsoft Excel. Dengan adanya tombol “Tambah Absen” dan “Simpan Absensi”, pengguna dapat mengelola data kehadiran dengan lebih efisien dibandingkan pencatatan manual. Secara keseluruhan, program ini merupakan contoh implementasi dasar pembuatan aplikasi administrasi sekolah menggunakan Python yang dapat dikembangkan lebih lanjut, misalnya dengan menambahkan fitur edit, hapus data, login pengguna, atau integrasi database.
